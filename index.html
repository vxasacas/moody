About the project

"Moody" es un proyecto nacido de la inspiración del Code Challenge 2023, diseñado para aliviar la carga de trabajo del equipo a la hora de crear tiquetes para QA

En un esfuerzo por demostrar un sólido conocimiento técnico básico, Moody se desarrolla utilizando tecnologías esenciales que incluyen HTML, CSS y JavaScript. Estas herramientas permitirán la creación de una interfaz efectiva y fácil de usar para poder crear comentarios de tiquetes.

En un principio la idea era la de crear la tercera opcion del challenge, que nos ayudaria a crear el proyecto, peroo la verdad es que lo vi muy dificil asi que me quede con la opcion de animar el formulario.
